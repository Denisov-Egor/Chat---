# Консольный чат для локальной сети (C++)

Многопользовательский чат с аутентификацией, комнатами, приватными сообщениями и администрированием.  
Работает в терминале, кроссплатформенный (Linux, macOS, Windows).

## Возможности

- Групповой чат с комнатами (каналами)
- Регистрация и вход по паролю
- Администратор управляет чатом из консоли сервера
- Личные сообщения (`/whisper`)
- Смена ника без переподключения
- Подсветка упоминаний + звуковой сигнал
- Ограничение попыток входа (временный бан по IP)
- Защита от флуда
- Кик, бан, мут через админ-консоль
- Логирование в `chat.log`
- Динамический лимит подключений
- Цветной вывод

## Структура проекта

| Файл | Назначение |
|------|-----------|
| `common.hpp` | Общие заголовки, константы, прототипы функций |
| `common.cpp` | Реализация общих функций (сокеты, отправка, баны, логирование, пользователи) |
| `server.cpp` | Серверная часть: приём подключений, аутентификация, обработка сообщений, админ-консоль |
| `client.cpp` | Клиентская часть: ввод ника и пароля, отправка/приём сообщений, подсветка упоминаний |
| `test.cpp`   | Юнит-тесты Google Test |

## Компиляция

Требуется компилятор с поддержкой C++11 (сервер/клиент) и C++17 (тесты), библиотека pthread.

### Сервер и клиент
```bash
g++ -std=c++11 -pthread -c common.cpp -o common.o
g++ -std=c++11 -pthread server.cpp common.o -o server
g++ -std=c++11 -pthread client.cpp common.o -o client
```

### Тесты
```bash
g++ -std=c++17 -pthread test.cpp common.cpp -lgtest -lgtest_main -o test_chat
```

## Запуск

### Сервер
```bash
./server <порт> [макс_клиентов]
```
Пример: `./server 12345 10`

### Клиент
```bash
./client <IP_сервера> [порт]
```
Пример: `./client 127.0.0.1 12345`

## Использование

### Команды клиента
- `/help` — справка
- `/list` — список участников комнаты
- `/whisper <ник> <текст>` — личное сообщение
- `/nick <новый>` — сменить ник
- `/join <комната>` — войти в комнату
- `/leave` — вернуться в `#lobby`
- `/rooms` — список комнат
- `/quit` — выход

### Администраторские команды (консоль сервера)
- `/help` — справка
- `/list` — список всех участников (с IP и комнатами)
- `/kick <ник>` — отключить
- `/ban <ник>` — забанить по IP на час
- `/mute <ник>` — запретить сообщения
- `/unmute <ник>` — разрешить
- `/deluser <ник>` — удалить учётную запись
- `/quit` — остановить сервер

## Примечания

- Пароли хранятся в `users.txt` как хеши `std::hash` (учебный проект)
- Для работы по сети откройте порт в брандмауэре
